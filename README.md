# CREA – AnaEE Website

Static build of the **CREA participation in AnaEE-ERIC** website.

## Overview
This repository contains the **compiled static version** of the website generated with Grav CMS.  
The original Grav source files (pages, templates, and configuration) are **not included**.

## Features
- Fully static website (HTML, CSS, assets)
- Optimized for fast loading and simple deployment
- Responsive layout with custom hero section

## Tech Stack
- Generated from [Grav CMS](https://getgrav.org/) (Quark theme)
- HTML / CSS / minimal JavaScript

## Website
- Temporary URL: https://anaee-crea-website.wasmer.app/  
- Production URL: _coming soon_

## Structure
- Static HTML pages
- CSS and media assets
